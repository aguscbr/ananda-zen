# Ananda Zen Landing Page

This is a landing page for Ananda Zen, a yoga school for kids in Buenos Aires.

## Tech Stack

- React
- TypeScript
- Vite

## Project Structure

```
src/
  components/
  hooks/
  App.tsx
  main.tsx
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The site is deployed on Vercel.
